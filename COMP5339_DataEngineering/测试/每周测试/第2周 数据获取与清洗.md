## 一、选择题

**1.** 数据质量中 "Default 问题" 指的是？

- A. 数据完全缺失
- B. 用占位符（如 0 或 99999）代替缺失值

**2.** 以下 Pandas 操作中，哪个用于删除含缺失值的行？

## 三、应用题

**8.** 你有如下 DataFrame：

| name | age | city |
|------|-----|------|
| Alice | 25 | Sydney |
| Bob | 99999 | None |
| Carol | NaN | Melbourne |

请写出 Pandas 代码，完成以下操作：
1. 将 `age` 列中值为 99999 的替换为 NaN
2. 用 0 填充 `age` 列的缺失值，然后转为 int 类型
3. 用 "Unknown" 填充 `city` 列的缺失值

---

## 参考答案

1. **B**
2. **C**
3. **D**（int 不支持 NaN，而 NaN 本身是 float 类型）
4. **C**
5.
   - **Believable**：数据是否可信（如异常暴增的数据）
   - **Value-Added**：数据是否有实际价值
   - **Relevant**：数据是否与分析目标相关
   - **Accurate**：数据是否准确（无错误）
   - **Interpretable**：数据是否易于理解（无歧义）
6. 问题：若 `age` 列含 NaN，直接转 int 会报错。修改：先填充再转型：
   ```python
   df['age'] = df['age'].fillna(0).astype(int)
   ```
7. `replace` 替换特定字符串值（如 `'N/A'`）；`fillna` 只填充 `NaN`（Python 的空值）。两者作用对象不同，若缺失值以字符串形式存在，需用 `replace`。
8.
```python
import numpy as np
df['age'] = df['age'].replace(99999, np.nan)
df['age'] = df['age'].fillna(0).astype(int)
df['city'] = df['city'].fillna('Unknown')
```
