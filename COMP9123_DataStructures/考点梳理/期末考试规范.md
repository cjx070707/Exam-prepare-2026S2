## 一、考试制度

| 项目 | 内容 |
|------|------|
| 权重 | 总评 **50%** |
| 形式 | 线下监考 |
| 及格 | 期末卷 **≥ 40%** |
| 练习卷条件 | 2h 写作 + 10min 阅读；RESTRICTED OPEN（计算器 + A4 双面笔记） |

### 可能出现的题型（整门课层面）

- **MCQ** — 概念、复杂度、操作追踪
- **Short answer** — 解释、对比、权衡
- **Algorithm tracing** — 给定输入逐步执行
- **Complexity analysis** — 伪代码 Big-O
- **Pseudocode writing** — 设计/修改算法

### 练习卷 / 大题风格（四道大题）

- **4 题 × 15 分 = 60 分**，**无 MCQ**
- 每题常拆 **(a)(b)(c)**，分值见练习卷
- 答题语言：**英文**（题干与作答）；复习笔记可用中英混排

## 二、课件要求的答题结构（必须记住）

### A. 算法大题 — *Question Structure*

1. **Main Idea** — plain English
2. **Pseudocode** *(maybe)*
3. **Correctness Argument** *(maybe)*
4. **Running Time Analysis** — line-by-line + Big-O

### B. 较小问 / 数据结构设计题

1. Describe the data structure
2. Step-by-step how it works
3. Analyse time complexity
